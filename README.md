# EcoLavoura - Futuro Sustentável

Projeto desenvolvido para o Concurso Agrinho 2026 - Categoria Programação Front-End.

## Tema

Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente.

## Objetivo

O projeto mostra, de forma interativa, que a produção agrícola pode caminhar junto com a preservação ambiental.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Funcionalidades

- Jogo educativo com canvas
- Pontuação
- Saúde da lavoura
- Tempo de jogo
- Quiz sustentável
- Feedback de respostas
- Layout responsivo
- Seção de créditos

## Como jogar

1. Clique em iniciar jogo.
2. Clique nos resíduos que caem na lavoura.
3. Evite clicar nos elementos naturais.
4. Mantenha a saúde da lavoura até o tempo acabar.
5. Responda ao quiz para testar seus conhecimentos.

## Organização dos arquivos

- `index.html`: estrutura do site
- `css/style.css`: estilos e responsividade
- `js/script.js`: lógica do jogo e quiz

## Créditos

Projeto desenvolvido com HTML, CSS e JavaScript, sem uso de bibliotecas externas.
