Sobre o projeto

O EcoLavoura foi desenvolvido com HTML, CSS e JavaScript para mostrar, de forma interativa, que a produção agrícola pode caminhar junto com a preservação ambiental.

O projeto trabalha conceitos como cuidado com o solo, preservação da água, redução da poluição, responsabilidade ambiental e sustentabilidade no agro.
