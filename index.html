Sobre o projeto

🌾

Campo produtivo

O projeto mostra a importância da agricultura para a alimentação, a economia e a vida das pessoas.

♻️

Sustentabilidade

O jogo incentiva atitudes como reciclagem, cuidado com o solo, preservação da água e proteção da biodiversidade.

💻

Programação

Foram utilizados HTML, CSS e JavaScript para criar uma experiência interativa com pontuação, tempo, colisão e quiz.

Como funciona

Objetivo do jogo

⚠️ Evite
